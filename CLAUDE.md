# Prometheus Practice

## 환경 정보
- **플랫폼**: EKS (Amazon Elastic Kubernetes Service)
- **설치 방식**: Helm (kube-prometheus-stack)
- **Prometheus 버전**: 2.x (LTS)
- **네임스페이스**: monitoring
- **스토리지**: EBS (PersistentVolumeClaim)
- **원격 저장소**: Grafana Mimir (옵션)
